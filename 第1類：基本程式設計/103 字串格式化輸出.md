## 設計說明：
請撰寫一程式，輸入四個單字，然後將這四個單字以欄寬為10、欄與欄間隔一個空白字元、每列印兩個的方式，先列印向右靠齊，再列印向左靠齊，左右皆以直線 |（Vertical bar）作為邊界。

## 輸入輸出：
輸入說明

```

I
enjoy
learning
Python

|  learning     Python|
|I          enjoy     |
|learning   Python    |
```
